Pseudocódigo
bruteForceClosestPoints(Points)
  dmin = ∞
  i1 = 0
  i2 = 0
  for (i = 0; i < n - 1; i++)
    for (int j = i + 1; j < n; j++)
      d = sqrt((Points[i].x - Points[j].x)^2 + (Points[i].y - Points[j].y)^2)
      if (d < dmin)
        dmin = d
        i1 = i
        i2 = j
  return Points[i1], Points[i2], dmin
